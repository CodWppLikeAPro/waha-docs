| **API**                                                             | WEBJS |      NOWEB       | GOWS |

| **Get profile picture** <br> `GET /api/contacts/profile-picture`    |  ✔️   |        ✔️        |  ✔️  |
| **Block contact** <br> `POST /api/contacts/block`                   |  ✔️   |                  |      |
| **Unblock contact** <br> `POST /api/contacts/unblock`               |  ✔️   |                  |      |
